# IDX-MACOS

A repository inspired by kmille36/idx-windows-gui on Github.

---

### Usage

Head torward to https://idx.google.com/ and import a project.

On the import screen, paste this repo url to the box and **ALWAYS** tick the mobile sdk support.

Your new workspace will automatically run `dev.nix` under `.idx` folder.

### Machine information

#### For workspace:
 - 8vCPU
 - 32GB of RAM

#### For virtual machine:
 - 4vCPU
 - 4GB of RAM
